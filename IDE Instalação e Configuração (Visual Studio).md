<h3>Testes <h3>

<h5>Arrange</h5>

organiza o método de teste de unidade, inicializa objetos e define o valordos dados que são passados para o método testado.

<h5>Act</h5>

*Invoca* o método 

<h5>Assert</h5>

verifica se a ação do método em teste se comporta conforme o esperado
